The Ultimate Guide to CSS Selectors

Selectors are the core of CSS, and writing a good CSS requires using them right. In this article, you will learn the essentials to make your styling clean and effective.

.class

Selects all elements, where the class atribute equals your value

See the Pen CSS Selectors: .class by Robert Wolf (@wolfrob35) on CodePen.

Probably the most popular CSS selector, because you can select multiple elements with the same class. However, to keep the html cleaner, use it only if no other selectors are available.

#id

Selects element with id attribute equals your value

See the Pen CSS Selectors: #id by Robert Wolf (@wolfrob35) on CodePen.

Unlike the .class selector, only one element can have the same attribute in whole site. Use it only if the id attribute already exists for different purpose (e.g. anchor links, JavaScript functionality).

*

Selects all elements

See the Pen CSS Selectors: * by Robert Wolf (@wolfrob35) on CodePen.

Great with combination with other selectors, such as element element or element>element

element

Selects all elements of the relevant type

See the Pen CSS Selectors: element by Robert Wolf (@wolfrob35) on CodePen.

Great for creating a general styles of the website, use it as much as possible.

element1 element2

Selects all element2 inside element1

See the Pen CSS Selectors: element element by Robert Wolf (@wolfrob35) on CodePen.

Useful for separate styling for different page sections. The element1 then selects an area, where the element2 styling will be applied.

element1>element2

Selects all element2 where the parent is a element1

See the Pen CSS Selectors: element>element by Robert Wolf (@wolfrob35) on CodePen.

Great for creating a general styles of the website, use it as much as possible.

selector, selector

Combines two selectors together

See the Pen CSS Selectors: selector, selector by Robert Wolf (@wolfrob35) on CodePen.

Perfect to make you CSS shorter and clearer. It is mostly used for reseting default browser styling.

There are the basic CSS3 selectors, used daily. To see the complete list, visit CSS3 selectors reference on W3C
