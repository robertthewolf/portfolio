How to: Custom Text Highlighter

Designing a website is about focusing on details. With CSS3, it is now possible to customize the user's highlighter tool. In this brief article, you will learn how to do it!

Create ::selection in CSS

The whole trick lies in the new CSS3 slectors. By using the ::selection, we can define styling only for content, that the user's highlights with the cursor.

::selection { css declarations; }

Write your custom style

Now, we can start to implement our own styling. Start by using the background-color tag to change the highlighter color. To create an inverse effect, you can play around with the color tag as well.

See the Pen CSS Selectors: ::selection by Robert Wolf (@wolfrob35) on CodePen.

Provide compatibility with vendor prefixes

The :selection CSS pseudo-element is now supported by 76.95% browsers. However, to provide compatibility with Firefox, we need to include -moz- vendor prefix:

::-moz-selection { color: white; background: black; } ::selection { color: white; background: black; }
